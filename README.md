## Programme qui utilise la CollectionViewSource pour cacher les " New Item Placeholder " dans ListBox et ComboBox 

Ce programme utilise la notion de filtrage et de CollectionViewSource pour cacher les lignes "New Item PlaceHolder" dans les ListBox, les ComboBox qui sont non éditables.

Ce phénomène apparait lorsque la même collection est liée à ces composants graphiques et à une datagrid éditable. Dès lors, la ligne prête à être encodée dans la DG apparaît également dans la ListBox et la ComboBox.
